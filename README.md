# FlowChat

Исходный код учебного приложения из книги **Яны Седовой «Асинхронный Kotlin»** (БХВ, 2026).

FlowChat — это Android-мессенджер, написанный с нуля в процессе чтения книги. В нем на практике показано, как работают корутины, Flow, Room, Retrofit и архитектурные паттерны современного Android-приложения.

## Как запустить

1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Запустите на эмуляторе или устройстве

Серверная часть развернута и работает — дополнительно ничего настраивать не нужно.

## Структура проекта

```
app/src/main/java/com/astfreelancer/flowchat2/
├── data/
│   ├── db/          # Room: сущности, DAO, база данных
│   ├── network/     # Retrofit: API, модели, обработка ошибок
│   ├── prefs/       # DataStore: хранение deviceId
│   └── repository/  # репозитории
└── presentation/
    ├── ui/          # экраны (Fragments, Adapters)
    └── viewmodel/   # ViewModel, UI-состояние
```

## Требования

- Android Studio Hedgehog или новее
- Kotlin 1.9+
- Android API 26+
